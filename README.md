## 📦 Project Summary

> A **pixel-perfect clone** of the Amazon homepage — replicating the iconic navbar, search bar, category menu, and product grid. Built using **only HTML & CSS** to sharpen layout precision and attention to detail.

---

## 🧩 What's Recreated

```yaml
Navbar:        ✅ Logo, Search Bar, Account, Cart icons
Sub-Nav:       ✅ All, Categories, Deals, Prime menu
Hero Banner:   ✅ Promotional carousel-style banner
Product Grid:  ✅ Category cards layout
Footer:        ✅ Back to top + multi-column footer
```

---

## 🎯 Why I Built This

Cloning Amazon helped me master:

- 🧱 **Complex Navbar layouts** with Flexbox
- 🎨 Matching **exact brand colors** (`#131921` & `#FF9900`)
- 📐 **Grid-based** product card layouts
- 🖱️ Realistic **hover & dropdown** interactions
